CardioLink Admin v3.9 LTS
==========================

Versión administrativa estable previa a CardioLink 4.0.

Novedades principales
- Perfil Administrador independiente del usuario Matías.
- Acceso económico reservado a Matías y Administrador.
- Secretaría conserva agenda, pacientes, caja diaria y conteos operativos, sin proyecciones ni aranceles.
- Convenios y aranceles configurables con vigencia.
- Cálculo de facturación estimada solo al presionar “Calcular estimación”.
- Ficha del paciente ampliada: sexo, localidad, dirección y provincia.
- Resumen con primera y última atención, cantidad de atenciones y último profesional.
- Alta de nuevas coberturas directamente desde la ficha del paciente.
- Calidad de datos ampliada y botón de cobertura incompleta corregido.
- Duplicados considerados únicamente por DNI.

Importante
El usuario Administrador debe existir también en Supabase Auth con el mismo nombre antes de @cardiolink.local. El perfil interno sugerido es “administrador”.
